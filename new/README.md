# 📸 VISION 高端摄影工作室网站

一个现代、专业的摄影工作室展示网站，支持在线作品展示、价格方案展示和项目接单功能。

## ✨ 主要功能

### 1. 首页展示
- 专业的 Hero 头部设计
- 导航栏快速访问各个功能区

### 2. 作品展示
- 精选作品图片库
- 不同类型摄影分类展示
- 悬停交互效果
- 数据统计信息展示

### 3. 服务项目
- 6种服务项目详细介绍
- 专业服务卡片展示
- 便捷的服务导航

### 4. 价格方案 ⭐
- 3种定价套餐（基础、标准、高级）
- 套餐功能详细列表
- 标准套餐突出显示

### 5. 快速接单 ⭐
- **微信咨询** - 一键复制微信ID
- **电话预约** - 直接拨号功能
- **在线表单** - 详细的项目信息采集

### 6. 联系方式
- 在线咨询表单
- 社交媒体链接

## 🚀 快速开始

### 安装
1. 克隆或下载项目文件
2. 在项目目录中找到 `index.html`

### 运行
```bash
# 方式1：直接在浏览器中打开 index.html
打开 d:\vscod pj\new\index.html

# 方式2：使用 Live Server（VS Code 推荐）
1. 在 VS Code 中安装 "Live Server" 扩展
2. 右键点击 index.html 选择 "Open with Live Server"
3. 自动在默认浏览器中打开

# 方式3：使用本地服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 💡 功能详解

### 接单流程
1. **用户发起** - 通过三种方式之一（微信/电话/表单）
2. **信息采集** - 收集项目详细信息
3. **团队响应** - 24小时内联系客户
4. **项目洽谈** - 确认细节和报价
5. **拍摄执行** - 完成项目
6. **后期交付** - 发送最终作品

## 🚀 部署建议

### 本地部署
1. 使用任何 HTTP 服务器运行项目
2. 确保支持 JavaScript

### 云端部署
推荐使用以下平台之一：
- **Vercel** - 免费，支持 HTML 项目
- **Netlify** - 免费，操作简单

1. 将项目上传到 GitHub
2. 在 Vercel 官网（https://vercel.com）注册
3. 导入 GitHub 项目
4. 自动部署完成
